# How to install

```
git clone https://github.com/juuun0/docker-qbindiff.git
cd docker-qbindiff
make && sh install.sh
```

Once installation is finished, updates your PATH for example: `source ~/.zshrc` or re-open terminal.

Then you can run `docker-qbindiff <options>`
